### datafilter.js

---
datafilter is javascript library to filter data from an array of objects (you might consider it like json but not actual json ;)).


#### Code Snippet

```
<script>
    const filteredData = new Filter ({
        data:data,
        value:"4"
    }).Search();
</script>

```

### Api

###### 1. data:

> type:array

###### 2. value:

> type:string

###### 3. searchparam:

> type:string
